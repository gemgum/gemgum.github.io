Kontes Robot Pemadam Api Berkaki

Kontes robot pemadam api berkaki merupakan kompetisi robotika yang diselenggarakan oleh Pusat Prestasi Nasional, Kementrian Pendidikan dan Kebudayaan Republik Indonesia. Peserta KRI adalah tim mahasiswa yang berasal dari berbagai Perguruan Tinggi dari seluruh penjuru Indonesia. Kegiatan KRI selalu berubah setiap tahunnya sesuai dengan kebutuhan riset dan pengembangan teknologi beberapa defisi pada tahun 2020 yaitu:

  1. Kontes Robot Pemadam Api Indonesia (KRPAI),
  2. Kontes Robot Sepak Bola Indonesia (KRSBI) Beroda,
  3. Kontes Robot Sepak Bola Indonesia (KRSBI) Humanoid,
  4. Kontes Robot Seni Tari Indonesia (KRSTI),
  5. Kontes Robot Tematik Indonesia (KRTMI),
  6. Kontes Robot ABU Indonesia (KRAI).

Kontes Robot Pemadam Api Berkaki memiliki gambaran sebuah robot berkaki akan diletakan pada sebuah arena yang menyerupai rumah dengan 4 buah ruang diaman salah satu ruangan terdapat sebuah sebuah lilin yang mewakili sebuah titik api. Robot harus dapat menemukan keberadaan titik api tersebut dan memadamkannya. Pemenang ditentukan berdasarkan waktu yang tercepat.

Kontes pemadam api berkaki Universitas Negeri Yogyakarta seperti namanya Gareng Punk atau hewan garengpung yang memiliki kaki 6 dan berbagai perangkat pembangun hanya beberapa bagian yang saya kembangkan karena sistem yang kompleks, berikut perangkat yang telah saya kembangkan untuk dapat menjalankan misi:

Dalam Kontes Robot Pemadam Api Berkaki robot diporgram untuk menyelesaikan misi dimana robot akan ditaruh dalam sebuah ruangan(home) dan robot diharuskan untuk mencari titik api lalu memadamkannya setelah itu robot diharuskan kembali ke ruangan pertama(home) untuk mendapatkan nilai terbaik. Ada bebrapa konfigurasi ruangan, peletakan home robot, peletakan api dankondisi kondisi lain posisi tersebut diacaka sesuai dengan undian jadi robot harus bisa mengetahui rute yang akan dilalui. Pada gambar berikut dapat dilihat konfigurasi ruangan dan posisi api.

    • Konfigurasi ruang dan posisi home robot

      • Setiap warna mebedakan konfigurasi ruangan yang berbeda dan simbol bulat digunakan untuk posisi awal mula robot diletakan.

    • Konfigurasi ruang dan posisi api yang akan dipadamkan

      • Setiap warna mebedakan konfigurasi ruangan yang berbeda dan simbol bintang digunakan untuk posisi api yang akan dipadamkan, sudah dipastikan posisi api tidak akan sama ruangannya dengan posisi home robot hal tersebut dilakukan agar robot dapat menjelajah seluruh ruangan.

    Berikut merupakan contoh rute yang dilalui robot pada aturan KRPAI 2019

    Jika dihitung kemungkinan algoritma yang harus dimasukan agar robot dapat mengetahui dari mana ruang yang dianggap home dan ruang yang ada api, sekitar 40 lebih kemungkinan yang harus ada pada robot.
